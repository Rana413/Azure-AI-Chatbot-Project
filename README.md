# Azure AI Chatbot Project

This project is a simple chatbot connected to Azure AI Language Service for sentiment analysis.

The chatbot can:
- respond to user messages
- analyze sentiment as positive, negative, or neutral
- show its capabilities
- handle empty or malformed input

This project was adapted from the idea of a traditional echo bot and extended by integrating Azure AI Language Service.

## Files
- chatbot_code/config.py
- chatbot_code/simple_chatbot.py
- requirements.txt

## Azure Service Used
- Azure AI Language Service
- Sentiment Analysis

## How to Run
1. Install requirements
2. Set environment variables for Azure endpoint and API key
3. Run `python simple_chatbot.py`
